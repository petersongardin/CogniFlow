# CogniFlow

Sistema de execução pessoal com IA para TDAH.

## Propósito

O CogniFlow é um assistente de execução para transformar intenção em entrega. O sistema recebe tarefas por texto ou áudio, fragmenta projetos amplos em blocos executáveis, agenda sessões de foco e realiza follow-up ativo até conclusão, replanejamento ou divisão da tarefa.

## Princípio central

O sistema nunca deve devolver uma tarefa ampla como se fosse executável. Toda tarefa precisa ser convertida em uma próxima ação concreta, com verbo, objeto claro, critério de término e duração adequada.

## Unidade mínima de execução

A unidade operacional inicial é um bloco TDAH de 25 minutos de foco + 5 minutos de pausa.
